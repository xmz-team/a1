# build a1?
No script has been built for the time being.
**If you need to compile a1 from the source code, please follow the steps below.**
1. Compile flock-ios and move to a1/a1/bin/
2. Compile bundle_pid and move it to a1/usr/bin/
3. Create the pack/var/jb directory, and copy and paste a1 to pack/var/jb/ (remember to move DEBIAN) and then use dpkg to package it into deb.
